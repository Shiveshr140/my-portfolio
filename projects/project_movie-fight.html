The Movie Fight web application is a dynamic movie comparison tool that empowers users to explore and compare key statistics of their favorite movies. Designed with a clean and responsive user interface, this project combines the power of the Open Movie Database (OMDB) API with an intuitive user experience.

Utilizes a sophisticated autocomplete feature for a seamless and efficient movie search experience. Enables users to compare two movies simultaneously. Fetches detailed information, including posters, titles, genres, and plots from the OMDB API.

Compares essential movie statistics such as awards, box office earnings, metascore, IMDb rating, and IMDb votes. Visual indicators highlight the superior values in each category for easy comparison.

Technologies used

  • Java Script
  • HTML
  • CSS
