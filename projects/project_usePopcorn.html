usePopcorn is a movie app built with React that helps you find and keep track of the movies you love and you can rate them.

This app leverages React.js for seamless state management, data fetching from the OMDB API, and integration with local storage to retain user preferences

Key features include a responsive search functionality, detailed movie information retrieval, and the ability to save and rate watched movies.

The application is optimized for performance with error handling and asynchronous operations, ensuring a smooth user experience.

Technologies used

  • React JS
  • OMDB API
  • HTML
  • CSS
