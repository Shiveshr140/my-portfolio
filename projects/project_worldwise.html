Developed an application where users can add cities according to their adventures and write a note about your adventure, showcasing my ability to handle diverse functionalities and integrate various features. If you're a wander enthusiast, this app is perfect for you.

You can add and share your adventures from the cities you've visited. With the integration of the React Leaflet library, you can visually track your footsteps on an interactive map.

Coding is extremely rewarding. As you learn, things start to click and make sense. You can join the dots of all the things that weren't quite clear before.

Technologies used

  • React JS
  • OMDB API
  • CSS
  • HTML
